# Development Lab Portfolio

A clean, responsive web development portfolio showcasing fundamental concepts in HTML, CSS, and JavaScript — built entirely from scratch without frameworks.

## Overview

This project serves as a structured learning journey through core web development concepts, progressing from basic HTML structure to a fully functional JavaScript application.

Each "lab" represents a focused exercise designed to build practical understanding of frontend development.

---

## Features

* Responsive layout using CSS Grid & Flexbox
* Modern dark-themed UI with custom CSS variables
* Modular lab-based structure for progressive learning
* Interactive components using vanilla JavaScript
* Final capstone project with localStorage persistence

---

## Project Structure

```
/
├── index.html        # Main portfolio page (this file)
├── lab1.html         # Basic HTML structure
├── lab2.html         # Forms and tables
├── lab3.html         # CSS box model
├── lab4.html         # Layout with Flexbox & Grid
├── lab5.html         # JavaScript fundamentals
├── lab6.html         # DOM manipulation
├── lab7.html         # Event handling
├── todo.html         # Final To-Do App (capstone)
```

## Learning Modules

### 01: Basic Structure

* Semantic HTML
* Headings and lists

### 02: Forms & Tables

* User input elements
* Data structuring

### 03: The Box Model

* Margin, padding, borders
* External stylesheets

### 04: Flexbox & Grid

* Responsive layouts
* Modern CSS techniques

### 05: Logic & Control

* Variables, loops, conditionals

### 06: Interaction

* DOM manipulation
* Dynamic UI updates

### 07: Event Handling

* User interactions
* Event listeners & timers

### Final Capstone: To-Do App

* Task creation & deletion
* Persistent storage using `localStorage`
* Mobile-friendly UI
